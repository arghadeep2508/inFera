# inFera — AI-Powered Data Analysis Platform

![Status](https://img.shields.io/badge/status-active-success)
![Frontend](https://img.shields.io/badge/frontend-Next.js-black)
![Backend](https://img.shields.io/badge/backend-FastAPI-green)
![ML](https://img.shields.io/badge/ML-Scikit--Learn-orange)

inFera is a full-stack AI-powered platform that transforms raw datasets into insights, predictions, and interactive analytics.

Built with a focus on real-world usability — combining data science, APIs, and frontend interaction into a single system.

---

## ✨ What Makes inFera Different

- Not just a model — a complete data system  
- Combines **EDA + ML + AI + UI** in one workflow  
- Designed for real interaction, not just output  

---

## 🚀 Features

- Automated exploratory data analysis (EDA)  
- AI-generated insights and pattern detection  
- Real-time machine learning predictions  
- Interactive “chat with your data” interface  

---

## 🧠 Tech Stack

**Frontend**
- Next.js  
- Tailwind CSS  
- Recharts  

**Backend**
- FastAPI  
- Python  

**Machine Learning**
- Scikit-learn  
- Pandas  
- NumPy  

---

## ⚙️ Local Setup

**Backend**
cd backend  
pip install -r requirements.txt  
uvicorn main:app --reload  

**Frontend**
cd frontend  
npm install  
npm run dev

## 🏗️ System Overview

inFera follows a modular architecture:

- Frontend handles visualization and interaction  
- Backend exposes APIs for data processing  
- ML layer performs prediction and inference  
- AI layer generates insights and responses  

---

## 💡 Key Insight

Building machine learning models is easy.  
Building systems where users can interact with data meaningfully is the real challenge.
